# 🧟 Creature Terminal Project

Projeto desenvolvido em **Lua** com o objetivo de simular uma ficha de criatura no terminal, utilizando formatação em texto, organização visual e barras de atributos.

O projeto exibe informações detalhadas sobre uma criatura inspirada no universo de jogos, incluindo descrição, aparência em ASCII e atributos.

## 🚀 Funcionalidades

- Exibição de nome da criatura
- Descrição formatada com quebra de linha
- Aparência em ASCII Art
- Informações adicionais:
  - Comportamento
  - Horário que aparece
  - Som característico
  - Drop
- Sistema visual de atributos com barras:
  - Ataque
  - Defesa
  - Vitalidade
  - Velocidade
  - Inteligência

---

## 🛠 Tecnologias Utilizadas

- **Lua**
- Terminal / CLI
- Strings multilinha
- Concatenação de strings
- Caracteres especiais (`\n`)

---

## 🎯 Objetivo do Projeto

Este projeto foi criado com foco em:

- Praticar lógica de programação
- Trabalhar formatação de saída no terminal
- Aprender manipulação de strings em Lua
- Criar interfaces visuais simples sem interface gráfica
